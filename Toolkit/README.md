## 📂 Projects

### ✅ DNA Purine Counter
A simple C program that counts the number of purines (A and G) in a DNA sequence.

**Concepts Used**
- Strings
- Arrays
- Pointers
- Loops
- Functions

---

### ✅ DNA Sequence Toolkit v1
A reusable toolkit for basic DNA sequence analysis.

**Features**
- Calculate DNA sequence length
- Count any nucleotide (A, T, G, C)
- Calculate GC Content
- Calculate AT Content

**Concepts Used**
- Functions
- Pointers
- Strings
- Modular Programming
- Code Reusability
